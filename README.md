# 🎙️ AI 播客

**粘贴一个链接，30 秒后开始听播客。**

不用录音棚，不用剪辑，不用约嘉宾。把任何一篇文章或一个想法，变成一期有对话、有碰撞、有节奏感的双人播客。

<br>

## 为什么需要它？

你一定遇到过这些场景：

> 📚 收藏夹里攒了 50 篇文章，但一直没时间看

AI 播客帮你把它们变成通勤路上的播客，用耳朵消化内容。

> 🎤 想做播客但没设备、没搭档、没时间剪辑

输入一个话题，比如「远程办公的未来」，30 秒后就有一期完整的双人播客。

> 📢 写了篇深度长文，想让更多人看到

一个链接转成音频，一键分发到播客平台，多一个内容触点。

> 🧠 看文字容易走神，听两个人聊更容易吸收

同样的内容，变成对话比独白更有温度，两个视角比一个更有深度。

<br>

## 30 秒上手

**第一步** — 打开 AI 播客网页

**第二步** — 粘贴文章链接，或输入一句话话题

**第三步** — 点击生成，边听边看实时字幕

就这么简单。

<br>

## 两种创作方式

### 🔗 链接转播客

粘贴任意网页链接 —— 公众号、知乎、掘金、博客……AI 自动读完全文，生成两位主播围绕文章展开的讨论。不是机械朗读，是真正的「聊」。

### 💬 话题转播客

没有现成文章？直接说你想聊什么：

- 「AI 如何帮助独立开发者提升效率？」
- 「为什么好产品都在拼情绪价值？」
- 「聊聊 Agent 和工作流的差别」

AI 联网搜索、组织观点、生成对话，一气呵成。

<br>

## 产品亮点

🎧 **边生成边听** — 不用等全部完成，音频实时推送，打开就能听

💬 **实时字幕** — 生成过程中展示每位嘉宾的对话，知道谁在说什么

🎭 **两组嘉宾风格** — 轻松日常选「咪仔 × 大壹」（男女对谈），深度访谈选「刘飞 × 潇磊」（双男声）

📥 **一键下载 MP3** — 生成完直接下载，转发或上传到播客平台

📱 **移动端友好** — 手机上粘贴链接，戴上耳机，通勤路上就能听

<br>

## 自己部署一个

想拥有自己的 AI 播客服务？项目完全开源，部署只需要几分钟。

### 你需要准备

- [火山引擎](https://console.volcengine.com/) 语音合成服务凭证（必需）
- [智谱 AI](https://open.bigmodel.cn/) API Key（可选，让链接模式的标题更智能）

### 一键部署到 Railway（推荐）

1. Fork 本项目到你的 GitHub
2. 在 [Railway](https://railway.app) 创建项目，连接仓库
3. 填写环境变量（见 `.env.example`）
4. 部署完成，拿到你的专属链接

### 本地运行

```bash
git clone https://github.com/CarlosLiOxO/AI-boardcast.git
cd AI-boardcast && npm install
cp .env.example .env  # 填写 API 凭证
node server.js         # 访问 http://localhost:3000
```

### 其他部署方式

项目同时支持 Netlify（前端）+ Railway/Render（后端）分离部署，以及 VPS + Nginx 自建。详见项目中的配置文件。

<br>

## 技术简介

极简架构，3 个运行时依赖，零构建工具：

- 后端 Node.js + Express + WebSocket，前端原生 HTML/CSS/JS
- 语音合成基于火山引擎 Podcast TTS，标题提炼基于智谱 GLM
- 内置安全防护：CSP / SSRF 防护 / 请求限流 / 跨域校验
